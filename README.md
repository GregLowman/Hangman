# Hangman

A Tkinter-based Hangman game with progressive hangman illustrations, 
clickable letter buttons, and a used-letters tracker.

## Files

| File | Description |
|------|-------------|
| `Hangman.py` | Main game — random word selection, letter buttons, progressive hangman images |
| `Hangman Pictures/` | PNG assets for each stage of the hangman drawing |

## Requirements

- Python 3.x
- `tkinter` (included with standard Python)
- `Pillow` — `pip install Pillow`

## Usage

```bash
python Hangman.py
```
